# Number-Guessing-Game
A guessing game where you try to figure out a randomly sellected number between 1 and 1000. Playable [here](https://p1x3l-w0lf.github.io/number-guessing-game/).

## About This Project
I made this game with html, javascript for logistics, and css for some basic styling.

 There is also a shop with boosters that you can buy to get hints for what the number is or get more points to spend in the shop.

## What I Got From This Project
I didn't really get any new skills from this one, but I did build up much more confidence with javascript and css!

## Modding
If you want to change the game's code you can...

• adjust structural webpage elements in index.html

• switch up the game logic and that kinda thing in script.js

• or change style.css to get the aesthetic of the game to your liking
